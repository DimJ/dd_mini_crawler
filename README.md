# dd_mini_crawler
A test crawler for PHP, used through a terminal

# Prerequisites
Linux system with the following:
- Git
- Composer
- PHP
- CURL

# Installation
- git clone https://github.com/DimJ/dd_mini_crawler.git
- cd dd_mini_crawler
- composer install

# Execution
- php main_script.php

# Modify list of products
Change the list of products in function get_file_urls (#60)
